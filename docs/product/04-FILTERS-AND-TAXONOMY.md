## URL Pattern for Filters

All filters are applied via query parameters on the listing page:
```
/restaurante-bucuresti?recommended=1&zona=centrul-vechi&bucatarie=italiana&pret=2
```

Multiple filters can be combined. Pagination uses `?p=N`.

**Note on counts:** Counts within a filter category (e.g., Venue Types) do NOT sum to the total (993) because venues can have multiple tags. For example, a venue can be both "Restaurant" and "Terasa" and "Pet friendly." The only category where counts sum correctly is **Star Rating** (518 + 162 + 5 + 0 + 0 + 308 = 993), since each venue has exactly one rating state.
