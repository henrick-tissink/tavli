# Product research & vision (pre-build)

These documents are **early, pre-build artifacts** — they predate the Tavli
codebase and do **not** describe the software as it exists today.

- `00`–`08` are **competitor research on [ialoc.ro](https://ialoc.ro)** (the
  Romanian reservation incumbent Tavli was conceived against): its platform
  architecture, consumer/business features, taxonomy, detail page, reservation
  flow, content strategy, and a design critique of where it falls short. Each is
  titled "— ialoc.ro"; they are a snapshot of the competitor circa April 2026,
  kept for reference. They are intentionally **not** updated to match Tavli.
- `09-TAVLI-VISION.md` is the **original product vision** — the name rationale,
  differentiation strategy, and an aspirational feature wishlist. It is a
  statement of intent, not a description of shipped behavior; many items have
  since been built (and some deliberately not).
